# WELCOME!
下載後打開<code>Start.bat</code>就可以開始使用了。

## 使用方法
1. 輸入啟動延遲時間(預設是10秒)
2. 在延遲時間內將畫面切換到Discord的狗狗指令專區
3. 等待

建議使用虛擬機或閒置電腦來跑這個程序，否則你需要固定每隔一小時將畫面切換回去。

## 注意!
* 你的電腦需要安裝 Java 8 以上才能執行這個程式
* <strong><em>請不要在其他文字頻道用這個機器人，被發現就直接處罰</em></strong>
