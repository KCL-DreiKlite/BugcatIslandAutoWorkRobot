# WELCOME!
本機器人的終極目標就是要讓大家共同富裕 :D

------------------------------------------

## 使用方法
1. 將檔案下載下來
2. 打開 `Start (for Windows).bat`
3. 按照畫面指示操作後按下 **Start Robot**
4. 將視窗切回Discord
5. 等待

**!! 建議使用虛擬機或閒置電腦來跑這個程序，否則你需要固定一段時間將畫面切換回去。**

## 注意!
* 你的電腦需要安裝 Java 8 以上才能執行這個程式
* ***請不要在其他文字頻道用這個機器人，被發現就直接處罰***
